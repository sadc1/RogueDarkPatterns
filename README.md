# Rogue Dark Patterns Game

This is a local browser-based game for CISC. 3650 focusing on dark patterns with user manipulation. The gameplay theme is a top-down rogue shooter, a 2D shooter game with round-by-round progression on a limited timer. There are substats to upgrade in order to kill more NPCs, and NPCs scale with the round.

## Controls
All controls can be found in the game but WASD for movement, P to pause, and mouse clicking on stats to upgrade.
